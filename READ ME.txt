- This is an Angular application named Booking Movies. 
- It's made for easy booking tickets for the latest movies in theaters.
- On the home page there is a short review of the movies that you can book ticket for,  and here you can find information about the availability. 
- If you click on the review button there is more information about each movie. 
- On MyBookings tab you can find information how many tickets you have recently booked. 
- This application works localy on localhost:1234.
